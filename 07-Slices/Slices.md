1. 定义
  array：固定长度的同类型元素集合
  slice：可变长度的同类型元素序列
2. 本质区别
  array 是固定大小的值
  slice 是对底层数组一段连续区域的引用/描述
3. 类型区别
  array 的长度是类型的一部分
  所以 [3]int 和 [4]int 是不同类型
  slice 的类型只关心元素类型，不关心长度
  所以写作 []int
4. 写法区别
  [3]int → array
  []int → slice
5. 使用场景
  长度固定 → 用 array
  长度经常变化 → 用 slice

[3]int 是数组
[]int 是切片，slice 类型只说明“元素是什么类型”，不把长度写进类型里。
slice 会说明 底层数据在哪里，从哪里开始，当前长度是什么，容量是什么
