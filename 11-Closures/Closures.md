Go 支持匿名函数， 并能用其构造 闭包。

所谓的闭包是指 函数可以“记住”它定义时所在作用域里的变量，即**内层函数用到了外层函数的变量**
哪怕外层函数已经执行结束了，只要这个函数还被保存、传递、调用，它仍然可以继续访问那些变量。
- 像带着状态的函数
- 每次调用，不一定从头开始
- 能操作之前的变量

```go
func outer() func() int {
    x := 0
    return func() int {
        x++
        return x
    }
}
```
1. 外层函数里定义一个变量
2. 外层函数里再定义一个内层函数
3. 内层函数里使用外层函数的变量
4. 然后把内层函数返回出去，或者保存起来

在循环里使用闭包时，要特别注意循环变量
- 每次循环，内层函数都引用了不同的变量 x
- 所以每次调用内层函数，都不会影响到其他循环的结果
